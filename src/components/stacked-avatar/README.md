# Stacked Avatar

Stacked Avatar lets you place Avatars on top of each other.

## Props

| Prop | Type | Description |
| ---- | ---- | ----------- |
| avatars | array | Place the desired avatars to be displayed in an array |

## Examples

  avatars={ [
    {
